云何 異生性 。 乃至廣說 。 問 何故 作 此 論 。 答 為 止 他 宗 顯 正義 故 。 謂 或有 執 欲界 見苦所斷 十 隨眠 是 異生性 。 如 犢子 部 。 彼 說 異生性 是 欲界繫 。 是 染污 性 是 見所斷 。 是 相 應 行 蘊 攝 。 為遮彼執 顯 異生性 是 三界 繫 。 是 修所斷 。 是 不 染污 。 是 不相應 行蘊所攝 。 或復有執 。 異生性 無實 體 。 如
